# Absensi Siswa

Website untuk absensi siswa dengan menggunakan NodeJS, data akan tersimpan kedalam file JSON yang ada di folder data.

## Teknologi yang digunakan

- HTML
- Javascript
- [Bootstrap]
- NodeJS
- ExpressJS
- EJS (View Engine)
